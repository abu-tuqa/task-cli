# task-cli
أداة مفتوحة المصدر تتبع فلسفة الحد الأدنى، صُنِعت لتكن خفيفة وفائقة السرعة مصنوعةً باللغة البرمجية لوا للتيرمنال. كثرت التطبيقات المشبعّة بالأدوات والبصريات المستثيرة التي لا تحتاجها؛ لذلك صُنِعت الأداة.

إن كنت شخصًا ترغب بأداة للمهام اليومية أو اللحظية، لا تريد أن تتفلسف عليك: هذه أداتك.

## الأوامر
### task new [نص]
لصنع مهمة جديدة؛ لا تحتاج للعلامات التنصيصية، الفراغات مسموحة. مثالًأ: task new drink water

### task list
لعرض المهام الحالية، التقدم الحالي مع توقيتات إدخال المهام مهمة بمهمة.

### task oops [رقم] [نص]
لتعديل مهمة تم إدخالها مسبقًا. لمعرفة الرقم يمكنك ببساطة استعمال *عقلك*، فإن المهمة الأولى رقمها 1، المهمة الثانية... وهكذا؛ أو استعمال `task list` لرؤية أرقام (أو معرّفات للمتعلمون) المهام الحالية.

### task finish [رقم]
لإنهاء مهمة ما حاليًا. **لا تُحذف المهام المكتملة**، بل يتم نقلها لملف `todo_done.txt` في جهازك حيثُ يمكنك الإطلّاع عليها لاحقًا لرؤية التقدم أو لمجرد أن بك حكة وتريد رؤية مافعلت في حياتك.
### task history
لرؤية المهام التي تم إكمالها سابقًا.
### task nuke
تبعثرت المهام في قائمتك وتريد التخلص من كل شيء؟ أكتب هذا الأمر وسيحذف جميع المهام الحالية بلا حفظ **وبدون أن يسألك**. تستعمله على مسؤوليتك، فهو ليس أمرًا مصنوع للاستعمال اليومي / المكثف.

## تثبيت الأداة
تثبيت الأداة أمرٌ بسيط. إن أردت تثبيتها بشكل عام في جهازك، أستعمل:
```
curl -s https://raw.githubusercontent.com/abu-tuqa/task-cli/main/task.lua -o task

chmod +x task

sudo mv task /usr/local/bin/
```
